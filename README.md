# rims_match
find your dream rims with Rims Match, You can draw or upload image to find a rims that similar to your dream rims.
# How to run
- Docker image<br>
  `docker pull koakattakoa/rims_match:latest`<br>
  `docker run -p 8000:8000`<br>
  open your browser to `localhost:8000`
- clone this repo<br>
  `pip install -r requirements.txt`<br>
  `python manage.py runserver`
